## 📝 **Deskripsi Singkat**

**Ravell Networks** adalah blog teknis modern yang dibangun menggunakan **React**, **TypeScript**, dan **Tailwind CSS**.
Dirancang untuk memberikan pengalaman membaca yang cepat, bersih, dan profesional — terutama bagi pembaca yang tertarik pada dunia **Networking**, **Cybersecurity**, dan **IT Infrastructure**.

Blog ini dilengkapi dengan berbagai fitur canggih seperti dark mode, TOC otomatis, skeleton loading, smart search, hingga efek animasi modern.

---

## 🚀 **Fitur Unggulan**

### ✨ **UI/UX & Tampilan**

* Desain tiga kolom ala *Holy Grail Layout*.
* Dark/Light mode dengan animasi lembut.
* Header adaptif (hide on scroll down, show on scroll up).
* Skeleton loading yang estetis.
* Responsif untuk semua perangkat.
* Hover dan transition yang halus.

### 📰 **Konten & Navigasi**

* Render Markdown secara penuh.
* Syntax highlighting otomatis.
* Table of Contents (TOC) dengan scroll-spy.
* Smart Search (desktop + mobile overlay).
* Breadcrumbs dinamis.
* Image zoom (lightbox).
* Pagination yang rapi.
* About Page profesional + portfolio mini.

### 📦 **Teknis**

* React Context untuk state global (SidebarContext).
* API client terpisah.
* Reading progress bar.
* Copy to clipboard untuk code blocks.

## ⚙️ **Instalasi & Penggunaan**

### 1. Clone Repo

```bash
git clone https://github.com/USERNAME_ANDA/NAMA_REPO_ANDA.git
cd NAMA_REPO_ANDA
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Jalankan Development Server

Jika menggunakan **Vite**:

```bash
npm run dev
```

Jika menggunakan **CRA**:

```bash
npm start
```

### 4. Buka di Browser

```
http://localhost:5173
```
